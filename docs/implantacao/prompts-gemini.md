# Prompts do Google Gemini — Entrega Final

> Este arquivo reúne os **6 prompts prontos** para você colar no
> [Google Gemini](https://gemini.google.com). Para cada um, tire um print da
> resposta e cole no ponto indicado do documento
> [`DOCUMENTO-IMPLANTACAO.md`](./DOCUMENTO-IMPLANTACAO.md).
>
> **Dica:** faça os prompts na ordem. As respostas do Gemini ajudam de verdade —
> e os prints comprovam o uso da IA, que é um requisito da entrega.

---

## Como usar

1. Abra https://gemini.google.com (logado na sua conta Google).
2. Copie um prompt abaixo e cole no Gemini.
3. Leia a resposta, ajuste se quiser, e **tire um print**.
4. Salve o print com o nome sugerido (ex.: `gemini-1-plano.png`).
5. Cole o print no local indicado do documento de implantação.

---

## Prompt 1 — Plano de implantação · (Seção 1.5)

```
Sou estudante de Ciência de Dados e vou implantar meu portfólio de projetos num
repositório GitHub público chamado portfolio-hub. Preciso de um plano de
implantação profissional cobrindo: organização do repositório, gestão de
usuários e permissões, políticas de segurança (branch protection, 2FA,
Dependabot), fluxo de colaboração com Pull Requests, integração contínua com
GitHub Actions e publicação em produção via GitHub Pages. Me dê um plano em
etapas, com boas práticas para cada uma.
```
